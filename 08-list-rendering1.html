item in items 형태의 문법을 사용. items는 보여줄 데이타의 배열, item은 반복되는 개별 항목의 별칭
배열 순환
  • {{parentMessage}} - {{ item.message }}
배열 순환, INDEX 참조
  • {{ index }} - {{ item.message }}

  • {{ key }}: {{ value }}

  • {{index}}. {{ key }}: {{ value }}
